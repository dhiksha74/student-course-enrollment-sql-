# Student Course Enrollment Management System

## Description
Mini project using SQL to manage students, courses, and enrollments.

## Tables
- Students (student_id PK, name, department, year)
- Courses (course_id PK, course_name, duration, fee)
- Enrollments (enroll_id PK, student_id FK, course_id FK, enroll_date)

## SQL Concepts Used
- Primary Key, Foreign Key
- CREATE, INSERT
- SELECT, WHERE, JOIN
- GROUP BY, COUNT

## Sample Queries
- List all students
- Students enrolled in a specific course
- Total students per course
- Department-wise student count
